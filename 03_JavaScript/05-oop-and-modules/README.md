# OOP and Modules

## Overview
This module introduces object-oriented patterns and file-level modular architecture in JavaScript.

## What You Learn
- Class design and constructors
- Encapsulation and method design
- Inheritance basics
- Reusable utility modules with `import/export`

## Files and Purpose
- `00-reference.md` — quick reference for OOP/module concepts
- `classes.js` — class-based account example
- `modules/mathUtils.js` — reusable utility exports
- `modules/main.js` — module import usage

## How to Run
```bash
node classes.js
node modules/main.js
```

## Practice Tasks
1. Add transfer method to account class.
2. Create `stringUtils.js` and import it into `main.js`.
3. Add basic input validation in class methods.

## Expected Outcome
You should be able to structure medium-sized JavaScript code using classes and modules.
