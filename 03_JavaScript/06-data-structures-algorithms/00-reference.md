# 00 — DSA Quick Reference

## Core Terms
- **Array**: Ordered list.
- **Stack (LIFO)**: Last in, first out.
- **Queue (FIFO)**: First in, first out.
- **Map**: Key-value structure.
- **Set**: Unique values only.

## Complexity Basics
- Array index access: `O(1)`
- Linear search: `O(n)`
- Hash map average lookup: `O(1)`

## When to Use
- Interview prep and performance-aware coding.
