# 00 — Advanced JS Quick Reference

## Core Terms
- **Closure**: Function remembers outer scope.
- **Hoisting**: Declarations moved to top during compile phase.
- **Prototype Chain**: Inheritance lookup mechanism.
- **call/apply/bind**: Set `this` context manually.
- **Currying**: Break function into chained unary functions.
- **Debounce/Throttle**: Rate-control event handlers.
- **Memoization**: Cache expensive results.

## When to Use
- Interviews, performance tuning, framework internals understanding.
