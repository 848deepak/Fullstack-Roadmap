# Advanced JavaScript Concepts

## Overview
This module covers the JavaScript internals and advanced patterns expected in interviews and senior-level debugging.

## What You Learn
- Closures, lexical scope, and hoisting
- Prototypes and prototype chain
- `call`, `apply`, `bind`
- Currying and memoization
- Debounce and throttle

## Files
- `00-reference.md`
- `01-closures-scope-hoisting.js`
- `02-prototypes-chain.js`
- `03-call-apply-bind.js`
- `04-currying-memoization.js`
- `05-debounce-throttle.js`

## How to Run
```bash
node 01-closures-scope-hoisting.js
node 03-call-apply-bind.js
```

## Expected Outcome
You should be able to explain runtime behavior and apply performance-oriented patterns correctly.
