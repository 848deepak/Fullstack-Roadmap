# 00 — DOM & Browser Quick Reference

## Core Terms
- **DOM**: In-memory tree representation of HTML.
- **Selector APIs**: `querySelector`, `querySelectorAll`, `getElementById`.
- **Event Listener**: Attach behavior on events using `addEventListener`.

## When to Use
- Update UI from user actions.
- Build interactive forms, lists, filters, and toggles.

## Key Patterns
- Event delegation for dynamic lists.
- Keep DOM queries cached when reused.
- Separate data logic from rendering logic.
