# Browser APIs

## Overview
This module introduces practical browser APIs used in modern frontend applications.

## What You Learn
- Storage APIs (`localStorage`, `sessionStorage`)
- Geolocation API basics
- Intersection Observer for viewport tracking
- Clipboard API for copy interactions

## Files
- `00-reference.md`
- `01-storage-api.js`
- `02-geolocation-intersection-clipboard.js`

## Run Notes
Run these examples in browser environment (not Node.js).

## Expected Outcome
You should be able to use browser-native APIs with basic permission and safety handling.
