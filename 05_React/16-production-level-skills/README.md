# Production Level React Skills

## Overview
This module focuses on the standards and skills needed to ship maintainable React applications.

## Topics Covered
- Scalable folder structure
- Clean code practices
- Reusable component strategy
- TypeScript with React
- Authentication with JWT
- Role-based access control
- Deployment (Vercel / Netlify)

## Files
- `00-reference.md`
- `folder-structure-guide.md`
- `clean-code-checklist.md`
- `typescript-react-starter.md`
- `jwt-role-based-access.md`
- `deployment-guide.md`

## Suggested Workflow
1. Apply folder structure guide.
2. Enforce clean code checklist in PR reviews.
3. Type core data contracts with TypeScript.
4. Secure auth flows and role checks.
5. Deploy with environment-safe configuration.

## Expected Outcome
You should be able to move from demo-level apps to production-ready engineering practices.
