# Data Structures and Algorithms (JavaScript)

## Overview
This module focuses on interview-relevant structures and complexity thinking in JavaScript.

## What You Learn
- Stack and queue behavior
- Map and Set usage
- Time complexity basics for common operations

## Files and Purpose
- `00-reference.md` — DSA quick lookup
- `stack.js` — LIFO implementation
- `queue.js` — FIFO implementation
- `map-set.js` — Map/Set practical usage
- `complexity-notes.md` — complexity cheat sheet

## How to Run
```bash
node stack.js
node queue.js
node map-set.js
```

## Practice Tasks
1. Implement a simple priority queue.
2. Add parentheses validator using stack.
3. Add duplicate-check utility using Set.

## Expected Outcome
You should be able to map problem statements to correct data structures and reason about complexity.
