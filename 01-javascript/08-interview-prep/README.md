# JavaScript Interview Prep

## Overview
This section helps you revise core-to-advanced concepts for interviews and technical discussions.

## How to Use
1. Read one topic daily.
2. Explain it in your own words.
3. Solve one coding problem for that topic.
4. Write trade-offs and edge cases.

## Included Resource
- `questions.md` — curated concept and reasoning prompts

## Core Interview Areas
- Scope and closures
- Event loop and async behavior
- Prototypes and class syntax
- Array/object transformations
- Time/space complexity analysis

## What You Learn
- How to explain JavaScript internals clearly
- How to reason about trade-offs and edge cases
- How to answer concept + practical interview questions

## Expected Outcome
You should confidently explain JavaScript internals and solve practical coding tasks.
