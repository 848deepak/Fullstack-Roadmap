# OOP in JavaScript

## Overview
This module focuses specifically on object-oriented programming patterns in JavaScript.

## What You Learn
- Constructor functions
- Classes and inheritance
- Encapsulation with private fields
- Polymorphism and static methods

## Files
- `00-reference.md`
- `01-constructor-functions.js`
- `02-classes-inheritance.js`
- `03-encapsulation-polymorphism-static.js`

## How to Run
```bash
node 01-constructor-functions.js
node 02-classes-inheritance.js
```

## Expected Outcome
You should be able to design class-based models and explain OOP principles in JavaScript context.
