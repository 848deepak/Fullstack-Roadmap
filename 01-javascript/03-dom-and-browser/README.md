# DOM and Browser APIs

## Overview
This module introduces browser-side JavaScript: selecting elements, handling events, and updating UI dynamically.

## What You Learn
- DOM structure and node selection
- `getElementById` and `querySelector`
- Content/style updates from JavaScript
- Event listeners and user interactions
- Event bubbling/capturing and delegation

## Files and Purpose
- `00-reference.md` — quick conceptual guide
- `index.html` — playground markup
- `styles.css` — basic styling for visual clarity
- `app.js` — task input and interactive DOM updates
- `event-flow-and-delegation.js` — bubbling, capturing, and delegation patterns

## How to Run
Open `index.html` in browser and test add/toggle task flows.

## Practice Tasks
1. Add a delete button for each task item.
2. Add completed/pending counters.
3. Use event delegation for delete and toggle actions.

## Expected Outcome
You should be able to build interactive browser UI with clean event handling.
