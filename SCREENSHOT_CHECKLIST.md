# Screenshot Checklist (Portfolio Ready)

Use this checklist before sharing your repository publicly.

## Required Screenshots
- [ ] Repository home page with clean README sections visible
- [ ] JavaScript module folder overview
- [ ] React module folder overview
- [ ] Topic coverage matrix section
- [ ] One mini project UI from JavaScript track
- [ ] One mini project UI from React track
- [ ] Testing output (React testing project)

## Naming Convention
- `01-repo-overview.png`
- `02-javascript-track.png`
- `03-react-track.png`
- `04-coverage-matrix.png`
- `05-js-mini-project.png`
- `06-react-mini-project.png`
- `07-testing-proof.png`

## Suggested Folder
Store images in: `assets/screenshots/`

## Quality Guidelines
- Use consistent browser zoom (100%)
- Hide personal/private tabs
- Keep terminal text readable
- Capture full section, not cropped snippets
