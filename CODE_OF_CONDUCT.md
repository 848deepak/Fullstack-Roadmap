# Code of Conduct

This project follows a simple standard: be respectful, supportive, and constructive.

## Expected behavior

- Use welcoming and inclusive language
- Respect different experience levels
- Give actionable, respectful feedback
- Focus on learning and collaboration

## Unacceptable behavior

- Harassment, personal attacks, or discrimination
- Abusive or insulting comments
- Spam or intentionally disruptive behavior

If needed, maintainers may remove harmful content to keep the community safe.
