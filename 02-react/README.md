## Skills Progression

- Routing + hooks + local state
- Context API and global app state
- UI libraries and dashboards
- Performance tuning strategies
- Testing with React Testing Library and snapshots
- Advanced React architecture and production engineering

## Suggested Practice

- For each project, create a `FEATURES.md` and add 3 enhancements
- Open a pull request for each enhancement branch to track progress
- Keep screenshots in project-level README files

## Coverage Checklist

- Core components and props
- Hooks and side effects
- State management patterns
- Form handling and validation
- API data flow and error handling
- Performance tuning and memoization
- Testing strategies
- Architecture and deployment planning
