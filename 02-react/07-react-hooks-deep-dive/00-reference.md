# 00 — React Hooks Quick Reference

## Hooks
- `useState`: local state
- `useEffect`: side effects + cleanup
- `useRef`: mutable ref without rerender
- `useContext`: global shared value
- `useMemo`: memoized derived value
- `useCallback`: memoized function
- `useReducer`: reducer-based state transitions

## Rule
Only call hooks at top level of React components/custom hooks.
