# React Hooks Deep Dive

## Overview
This module provides focused examples for each core React hook and lifecycle behavior.

## What You Learn
- Stateful logic with `useState`
- Side effects and cleanup with `useEffect`
- Derived state with `useMemo`
- Stable callbacks with `useCallback`
- Mutable references with `useRef`
- Context usage with `useContext`
- Complex transitions with `useReducer`
- Mount/update/unmount lifecycle flow
