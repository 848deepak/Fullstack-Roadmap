# State Management: Advanced Notes

## Lifting State Up
- Move shared state to closest common parent.

## Context API
- Good for medium complexity and shared global settings/auth.

## Redux (Basics to Advanced)
- Central store and predictable updates.
- Use slices/selectors/middleware for scalability.
- Add async handling with thunks.

## Zustand (Optional)
- Lightweight store with minimal boilerplate.

## Recoil (Optional)
- Atom/selector model for fine-grained state graphs.
